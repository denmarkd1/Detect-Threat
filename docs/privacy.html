DT Scanner is built as a defensive mobile security utility. This policy explains what data the app accesses and how that data is handled.

Data Processing Model

  • Current version is on-device only.
  • No account signup is required.
  • Scan history is stored locally on the user's device.
  • No cloud sync is enabled in this version.

What The App Accesses

  • Installed app metadata relevant to local watchdog scanning.
  • Accessibility and device-admin state indicators for risk detection.
  • Notification permission status for alert display.

What The App Does Not Do

  • Does not sell personal data.
  • Does not share personal scan output to third parties in this version.
  • Does not provide a guarantee that a device is fully compromise-free.

Future Updates

If optional cloud backup or diagnostics are added in future versions, users will be asked for clear, explicit opt-in before any non-local data transfer begins.
